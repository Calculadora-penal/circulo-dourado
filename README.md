# 🎯 Círculo Dourado — Calculadora Penal

Aplicação interativa baseada no conceito do **Círculo Dourado**, utilizada para apresentar de forma clara e estratégica um projeto de Engenharia de Software voltado ao domínio jurídico.

🔗 **Acesse a aplicação:**  
https://calculadora-penal.github.io/circulo-dourado/

---

## 📌 Sobre o projeto

Este projeto aplica o modelo do **Golden Circle (Simon Sinek)** para estruturar e comunicar um sistema de cálculo de dosimetria penal.

A interface é composta por camadas interativas que representam:

- **Por quê?** → O propósito do sistema  
- **Como?** → O processo e as regras aplicadas  
- **O quê?** → As funcionalidades entregues  
- **Quem?** → Os usuários e stakeholders  

Cada camada pode ser explorada dinamicamente, exibindo informações detalhadas sobre o projeto.

---

## 🧠 Objetivo

Demonstrar como conceitos de **Engenharia de Software** podem ser utilizados para:

- Resolver problemas reais do sistema jurídico  
- Reduzir erros manuais em cálculos penais  
- Melhorar a transparência e compreensão de regras complexas  
- Criar interfaces intuitivas e informativas  

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- SVG (para visualização interativa)

---

## 🎨 Funcionalidades

- Visualização em camadas (círculos concêntricos)
- Interação por clique
- Exibição dinâmica de conteúdo
- Design minimalista e responsivo
